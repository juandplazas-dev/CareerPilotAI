# 🚀 CareerPilot AI

> Find smarter. Apply better. Grow faster.

CareerPilot AI es una plataforma inteligente para la gestión de la carrera profesional. Su objetivo es ayudar a los candidatos a encontrar mejores oportunidades laborales mediante automatización, inteligencia artificial y análisis inteligente de vacantes.

## ✨ Características

- 🔍 Búsqueda automática de vacantes.
- 🤖 Análisis de compatibilidad con IA.
- 📄 Generación de CV ATS mediante CareerOps.
- ✉️ Generación de cartas de presentación.
- 📊 Dashboard con estadísticas.
- 📅 Seguimiento de postulaciones.
- 🎯 Recomendaciones de mejora profesional.

## 🛠️ Tecnologías

- Python
- Git
- GitHub
- SQLite
- Streamlit
- Gemini AI
- CareerOps
